# resource-pc-cnweb

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖各类 PC 端网页资源。这些页面不针对任何单个域名或具体网站，仅作为静态资源存档使用。

## 项目简介

- 归档对象：独立、完整的 HTML 文件及相关资源
- 用途：提供可直接在浏览器中打开的页面存档
- 特点：不依赖特定域名、不追踪访问来源、不包含商业推广内容

## 目录结构

```
resource-pc-cnweb/
├── pages/          # 存放各个 HTML 页面文件
├── assets/         # 页面所需的样式、脚本、图片等资源
├── README.md       # 本说明文件
└── LICENSE         # 仓库许可（如有）
```

各子目录内文件按归档时间或主题分类，请参见对应目录中的说明文件。

## 页面归档说明

- 所有页面均为历史快照或独立制作，不保证与原始来源完全一致。
- 归档时尽可能保留页面原始结构与功能，但可能移除或替换外部链接、追踪脚本等。
- 页面中出现的任何商标、品牌、标识等均归其各自权利人所有，本仓库仅为存档用途。

## 维护说明

- 本仓库不定期更新，内容可能包含多个时间段的存档。
- 不承诺对归档页面进行持续更新或兼容性修复。
- 如发现页面存在明显错误或安全问题，欢迎提交 Issue 或 Pull Request。

## 使用方式

1. 克隆或下载本仓库到本地
2. 在浏览器中直接打开 `pages/` 目录下的 `.html` 文件
3. 或使用任意静态文件服务器托管整个目录
